### Align

参数说明

| 类型        | 默认               | 示例                          |
| ----------- | ------------------ | ----------------------------- |
| `Alignment` | `Alignment.center` | `align(Alignment.centerLeft)` |

#### Native

```dart
Align(
  child: Text("some text"),
);
```

#### Kit

```dart
Text("some text").align();
```
